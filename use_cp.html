Using Chapter Points

So you've got some Chapter Points (CP) and you want to spend them? Well, you've come to the right place. CP can be spent on the following things at the following costs. Some of these costs are set by the international organization, some are set by the chapter. If you have any questions about anything on this list, or would like to buy some of these, please feel free to contact logistics.
